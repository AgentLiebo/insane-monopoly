# Art Asset Organization

This folder is reserved for production art assets used by the standalone Unity game.

- `Board/` — board meshes, tile materials, deed/card art, VFX textures.
- `Tokens/` — player token meshes, pawn materials, animation clips.
- `../Prefabs/` — assembled gameplay prefabs once placeholder primitives are replaced.

The current prototype generates placeholder meshes procedurally at runtime so the project remains buildable without paid or copyrighted assets.
